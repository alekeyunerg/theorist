## 📄 What is theorist?

theorist is a simple application designed to help you keep track of your project's core ideas and guides. It stores key information about how each part of your project should work. If you manage multiple projects or code repositories, theorist helps you keep all your important notes organized in one place. 

You do not need to understand programming to use it. theorist works by creating and maintaining clear documents that explain how your project works. This makes it easier for anyone who works on your project to follow the same rules and ideas.
